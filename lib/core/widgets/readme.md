# Reusable Widgets Library

This directory contains professional, reusable widgets that follow Flutter best practices.

## Available Widgets

- `app_scaffold.dart` - Standard app structure with safe area
- `custom_app_bar.dart` - Professional app bar with customization
- `loading_indicator.dart` - Animated loading widget
- `error_widget.dart` - Error state display
- `section_header.dart` - Common section headers
- `stat_card.dart` - Stats display cards
